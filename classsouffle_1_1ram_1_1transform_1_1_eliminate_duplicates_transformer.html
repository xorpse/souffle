Detailed Description

Eliminates duplicated conjunctive terms.

For example ..

QUERY
...
IF C1 /\ C2 /\ ... /\ CN
...

will be rewritten to

QUERY
...
IF C2 /\ ... /\ CN then
...

assuming that C1 and C2 are equal.
